1. Buatlah sebuah fungsi untuk menjumlahkan nilai ganjil pada sebuah list.
Contoh :
	List : 1, 2, 3, 4, 5		List : 1, 3, 4, 6, 7 
	JumGanjil(L) : 9		JumGanjil(L) : 11

2. Buatlah sebuah prosedur untuk mengganti nilai dari elemen terakhir dengan
menambahkan nilai pada elemen terakhir sebesar X.
Contoh :
	List : 1, 2, 3, 4, 5
	GantiLast(L,X) X=10
	Output : 1, 2, 3, 4, 15


3. Buatlah sebuah fungsi untuk menentukan apakah nilai tengah dari sebuah list bernilai genap.
	- mengembalikkan true jika nilai tengah bernilai genap
	- mengembalikkan false jika nilai tengah bernilai ganjil.
Contoh :
	List : 1, 2, 3, 4, 5 ====> NilaiTengahGenap(L) : 0 
	List : 2, 4, 6, 8, 10 =====> NilaiTengahGenap(L) : 1

Penilaian
Kerjakan No. 1&2 dengan benar Poin 100
Kerjakan No. 3 saja dengan benar Poin juga 100
